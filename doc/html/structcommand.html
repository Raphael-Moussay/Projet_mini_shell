Detailed Description

Représente une commande et ses arguments.

  • name : nom de la commande (ex: "ls").
  • arg : tableau d'arguments, terminé par NULL.
  • nb_arg : nombre d'arguments (sans compter le NULL final).
  • separator : caractère séparateur qui a suivi la commande ('&', '|', '>', ';' ou '\0').
  • next : pointeur vers la commande suivante dans la chaîne.
