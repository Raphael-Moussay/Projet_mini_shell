Detailed Description

Représente un job en arrière-plan.

  • id : identifiant du job (affiché entre crochets).
  • pid : PID du processus fils associé.
  • arg : copie des arguments de la commande lancée.
  • nb_arg : nombre d'arguments (sans compter le NULL final).
  • status : état actuel du job.
  • next : pointeur vers le job suivant dans la liste chaînée.
